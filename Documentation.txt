This a simple client-server chatting application.It works only one time.The client side code is in windows application and the server side code is running on console. both of them are written in c#. First, the server code should run,then clinet code should run. User can enter any arithmatic expressin with(+/-*) and two operands, for example (6+2),after clicking the button, the expressin will send to server. the server will split the input to the members and it will recognize the oprator.so, It will calculate the output and send the result to client. at the end, the result will be shown within a label in client side.
